## 기능 구현 목록
- [ ] :  체스 게임 시작 문구를 출력한다.
- [ ] : 시작 명령어를 입력 받는다.
  - [ ] : start, end 가 아니면 예외를 터트린다.
  - [ ] : start 입력 시 게임을 시작한다.
  - [ ] : end 입력 시 게임을 종료한다.
- 
- [ ] : 보드판을 초기화한다.
  - [ ] : 소문자, 대문자로 팀을 구분한다.
  - [ ] : 각 팀은 총 16개의 말을 갖는다.
    - [ ] : 폰 8개, 룩 2개, 나이트 2개, 비숍 2개, 퀸 1개, 킹 1개
  

- [ ] : 말 객체(Unit)
  - [ ] : 위치, 팀
- [ ] : 위치 객체(Position)
  - [ ] : row, colume Enum을 갖는다.
